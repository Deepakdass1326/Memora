# 🧠 Memora

**Memora** is an intelligent, AI-powered knowledge operating system and "second brain." It allows users to seamlessly save, auto-tag, and meaningfully reconnect their notes, bookmarks, videos, and internet findings in one unified, scroll-free workspace.

Built with performance and aesthetics in mind, Memora dynamically links concepts via a Knowledge Graph and utilizes multi-provider LLM routing (Gemini, Cohere, Mistral) to contextually tag and summarize information.

### ✨ Features
- **Intelligent Knowledge Graph:** Visualizes connections between notes, articles, and bookmarks using automated context discovery.
- **Smart Search Engine:** Unified search that queries your local library, YouTube metadata, and direct DuckDuckGo/Web results simultaneously without leaving the app.
- **Multi-LLM Routing:** Background asynchronous workers that automatically categorize, tag, and summarize content using optimal AI models.
- **Chrome Extension Sync:** Capture content from anywhere on the web directly into your workspace.
- **Lightning Fast:** Powered by MongoDB aggregation pipelines, Redis-backed task queues, and automated payload compression.

<br>

**Stack:** MongoDB, Express.js, React (Vite), Node.js, BullMQ, Redis, Tailwind CSS/Vanilla CSS, Framer Motion.
